# E-commerce Website (Static)

This folder is intentionally **separate** from `crm-project/` and does not share any PHP includes or database code.

## How to view
- If you use XAMPP and `Ssweb` is inside `htdocs`, open `http://localhost/Ssweb/ecommerce-website/`
- Or open `index.html` directly in a browser.

## Pages
- `index.html` (Home)
- `products.html` (Product listing)
- `product.html` (Product details – static demo)
- `cart.html` (Cart – static demo)
- `checkout.html` (Checkout – static demo)
